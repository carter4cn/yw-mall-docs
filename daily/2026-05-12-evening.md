## 🐛 修的坑：依赖 hoist

**症状**：pnpm build 报 `Rollup failed to resolve import "hookable" from @vue/devtools-kit/dist/index.js`，加 hookable 后又报缺 `birpc`。

**根因**：pnpm 11 默认严格 hoist，`@vue/devtools-kit` 的 peer/transitive deps 不会被自动 hoist 到 root。Vite 在生产构建（vue-prod-devtools 默认包含）时仍 import devtools-kit，链式触发解析失败。

**修法**：直接把 `hookable` + `birpc` 加成顶层 dependencies（不污染逻辑代码）。`pnpm install` → build 通过。

> CLAUDE.md 说"不要加 build.rollupOptions.external for wot-design-uni"，正好与此呼应：解决依赖问题的对路方式是补 dep，不是 external。
